## Getting Started

Browse the `skills/` directory for ready-to-use skill examples that demonstrate Nero's capabilities.

## Contributing

Have a useful example? Submit a PR! Examples should:
- Be well-documented
- Follow Nero conventions
- Include clear use cases
- Be tested (where applicable)
